# USDai Contracts

## Usage

```shell
$ git clone --recursive https://github.com/metastreet-labs/metastreet-usdai-contracts.git
```

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Submodules

### Update

```shell
$ git submodule deinit --force .
$ git submodule update --init --recursive
```
